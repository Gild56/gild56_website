Quiz Master

Quiz Master is a simple Kivy quiz app. You can collect points, presonalize all for your taste, listen your favorite music and much more!

EXE is a Windows executable format easy to use. You have only one file that you can just open by clicking on it.

ZIP is a compressed archive. First, you need to "unzip" the file. Then, you need to install python and run the quizmaster/command/play.vbs file.

Valorant Randomizer

Valorant Randomizer is a custom fanmade program on Kivy to use while playing Valorant for randomize your character and weapons.

EXE is a Windows executable format easy to use. You have only one file that you can just open by clicking on it.

ZIP is a compressed archive. First, you need to "unzip" the file. Then, you need to install python and libs used in the project and run the valorant_randomizer/app.py file.
